DP260
产品介绍：

                DP260 

 

DIGITAL CROSSOVER SYSTEM PROCESSOR
专业数字式分频器/系统处理器
适用范围：
流动演出、体育场所、剧场、礼堂、多功能厅等高水准声音要求的场所。
功能特点：
DP系列基于DSP技术的音箱处理器，高性能24位高精度AD/DA，DSP低失真，大动态。DS系列是2输入，4/6输出，包括5种配置模式。每一款都包括输入增益控制，每一个通道都包括独立的分频限制器。5段参量均衡，延时，输出增益及相位控制，参数加锁，避免误操作。每一组参量均衡有31个（ISO）频率，-12dB到+12dB的增益。每一组Q值从0.5到10，并提供Hi-shelf，Lo-shelf选项功能。每个通道的高通、低通配置6dB,12dB，18dB，24dB，48dB的巴特沃斯（Butterworth）、宁克一锐（Linkwitz riley）贝塞儿（Bessel）频响曲线。通过RS232接口连接PC软件实时控制。
技术参数：
• 输出最大电平pp=4V平衡，Vpp=7.6V非平衡
• 频率响应：20Hz~20.2KHz
• 失真：0.01%(THD)
• 最大延时：-7ms
• 滤波器
• 频率（高通）：＜10Hz~16.0KHz
• 频率（低通）：35Hz~22.0
DP260 SERIES
DIGITAL CROSSOVER SYSTEM PROCESSOR
专业数字式分频器/系统处理器
Scope of application :
High standard occasions such as mobile performance,stadium,theater,lecture hall
andmulti functional meeting rooms.
FEATURES：
The DP series is based on DSP technology speaker processor, high-performance AD/DA, 24 bit high precision, DSP low distortion, large dynamic. DS series is 2 input, 4/6 output, including 5 types configuration mode. Each includes input gain control, and each channel is including independent frequency division limiter. 5 segment of parametric equalizer、delay、output gain and phase control, the parameter is locking to avoid the wrong operation. Each parametric equalizer has 31 (ISO) frequency, it is from -12dB to +12dB of gain. And each Q value of group is from 0.5 to 10, also provide the option function of Hi-shelf and Lo-shelf. Each channel configuration of high pass and low pass is 6dB,12dB，18dB，24dB，48dB of Butterworth、Linkwitz riley and Bessel Frequency Response Curve. It through RS232 interface to connect with PC software to control.
SPECIFICATIONS：
• Maximum output electronic level: pp=4V, balance, Vpp=7.6V unbalance
• Frequency response: 20Hz~20.2KHz
• Distortion: 0.01%(THD)
• Maximum delay: -7ms
• Filter
• Frequency(High pass): ＜10Hz~16.0KHz
• Frequency(High pass): 35Hz~22.0
